##  Overview
This machine learning project predicts passenger survival on the Titanic disaster. The model analyzes various features like age, gender, passenger class, and fare to determine survival likelihood with 82% accuracy.

##  Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA) 
- Multiple Machine Learning Models Comparison
- Feature Importance Visualization
- Model Performance Evaluation

## Technologies Used
- **Python 3.x**
- **Pandas & NumPy** - Data manipulation
- **Scikit-learn** - Machine Learning
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter Notebook** - Interactive analysis

##  Results
| Model | Accuracy | Precision | Recall |
|-------|----------|-----------|--------|
| Random Forest | 82% | 81% | 78% |
| Logistic Regression | 79% | 77% | 75% |
| Decision Tree | 80% | 79% | 76% |
